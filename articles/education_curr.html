Install VistA - Follow the instructions on this page. The first three listed will be of use for installing VistA. The other instructions will be used later in this curriculum. Developed by the VA eHealth University this image includes patient data for an added jump-start.

CPRS Configuration - Learn how to configure CPRS with this very informative setup guide. This Brief Overview of CPRS video is a good introduction to CPRS, along with this catalog of CPRS training.
Learn how to connect printers to VistA - This site provides instructions on setting up printers in VistA. Calatlog of VistA Basics - The "back-to-basics" series of myVeHU Campus training videos and powerpoints will teach many of the basics becoming familiar with the VistA EHR.
Learn how to configure a clinical package - Configure a clinical package such as pharmacy, lab, or radiology. Learn how to use Fileman for Sorting and Printing - This getting started manual for Fileman includes 10 chapters to help get yourself acquainted. In addition, this video and powerpoint on Fileman Basics will help get you started.
Learn how to configure HL7 - Use this extensive guide for getting started with HL7 and more.  
Learn how to use Fileman for Sorting and Printing - This getting started manual for Fileman includes 10 chapters to help get yourself acquainted. In addition, this video and powerpoint on Fileman Basics will help get you started.  
Reporting using KB-SQL - Commercial product for those interested in reporting using SQL.  
M Programming - Various resources available from the OSEHRA website to learn about M programming.  
VistA Programming - Resources also from the OSEHRA website to learn about programming in VistA. Additional specified training can be acquired here.  
Delphi Programming in CPRS - Documentation for programming against VistA using Delphi. More help can be found here.  
Newer Technologies Programming in VistA - No training material available; must consult documentation for various projects.  
International Users - VistA Internationalization is an ongoing project. For a background in VistA Internationalization, this powerpoint will cover the basics, while the current Plan VI Project Group Page will show the most up-to-date progress in this area.
